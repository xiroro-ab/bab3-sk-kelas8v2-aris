Cara Kerja Komputer

🔄 Prinsip Dasar Kerja Komputer

Komputer bekerja berdasarkan prinsip Input-Process-Output (IPO). Setiap operasi komputer, dari yang sederhana hingga kompleks, mengikuti siklus ini untuk mengubah data mentah menjadi informasi yang berguna.

📥

INPUT

Data masuk melalui perangkat input

⚙️

PROCESS

CPU memproses data sesuai instruksi

📤

OUTPUT

Hasil ditampilkan ke pengguna

🧠 Siklus Instruksi CPU

CPU menjalankan instruksi dalam 4 tahap berulang:

FETCH
Ambil instruksi
DECODE
Terjemahkan
EXECUTE
Jalankan
STORE
Simpan hasil
